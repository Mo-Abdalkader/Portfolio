Linear Regression:
The Model That Started Everything

The simplest ML model is also the most important to understand deeply. Linear regression teaches you loss functions, optimization, overfitting, and feature engineering — all at once.

← Back to Blog ☰ Categories

In 1886, Francis Galton studied the heights of parents and their children.
He noticed something: tall parents tend to have tall children — but shorter
than themselves. Short parents tend to have taller children — but shorter
than average. Heights "regress toward the mean."

He drew a line through his data. That line became the most important tool
in the history of data analysis.

01 — What Linear Regression Does

Linear regression answers one question: given some input features, what number do I predict as output? It finds the best straight line (or hyperplane in multiple dimensions) through your data that minimizes prediction error.

Examples: predict house price from size + bedrooms. Predict a student's exam score from study hours. Predict tomorrow's temperature from today's. Any time your output is a continuous number, linear regression is your first tool to reach for.

📏 Analogy

Imagine plotting every house sale in a city as a dot on graph paper — horizontal axis is size in m², vertical axis is price. Linear regression draws the single straight line that comes closest to all those dots simultaneously. Once you have that line, predicting the price of a new house is as simple as finding its size on the x-axis and reading off the y-axis.

02 — The Math (Made Friendly)

The model has the form: ŷ = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

Where x values are your input features (size, bedrooms, location score), w values are weights the model learns (how much each feature matters), and b is a bias term (the baseline prediction when all features are zero).

Training finds the w and b values that minimize the Mean Squared Error (MSE) — the average of squared differences between predictions and true values. Squaring serves two purposes: it makes errors positive (so negative and positive errors don't cancel), and it penalizes large errors much more than small ones.

What it predicts
A continuous number (price, temperature, score)
Loss function
Mean Squared Error (MSE)
What it learns
Weights (w) and bias (b)
Key metric
R² (coefficient of determination)

03 — What R² Tells You

R² (R-squared) measures how much of the variance in the target your model explains. R² = 1.0 means perfect predictions. R² = 0.0 means your model is no better than just predicting the average every time. R² can be negative — meaning your model is actively worse than the average baseline.

An R² of 0.85 on house prices means your model explains 85% of the variation in prices across houses. The remaining 15% is unexplained — perhaps due to unmeasured features (neighborhood reputation, renovation quality) or genuine randomness.

04 — Code: From Scratch vs sklearn

import numpy as np

# Dataset: hours studied → exam score
X = np.array([1,2,3,4,5,6,7,8], dtype=float)
y = np.array([45,50,58,61,68,74,79,84], dtype=float)

# From scratch: gradient descent
w, b, lr = 0.0, 0.0, 0.01
for _ in range(5000):
    pred = w * X + b
    w -= lr * np.mean(2 * (pred - y) * X)
    b -= lr * np.mean(2 * (pred - y))
print(f"Scratch → w={w:.2f}, b={b:.2f}")  # w≈5.24, b≈39.3

# sklearn — same result, one line
from sklearn.linear_model import LinearRegression
model = LinearRegression().fit(X.reshape(-1,1), y)
print(f"sklearn → w={model.coef_[0]:.2f}, b={model.intercept_:.2f}")
print(f"R² = {model.score(X.reshape(-1,1), y):.3f}")  # → 0.996

05 — When Linear Regression Breaks

  • When the relationship isn't linear. Adding a bedroom to a 30m² apartment matters hugely. Adding one to a 500m² mansion barely affects the price. A line can't capture this. You need polynomial features or a non-linear model.
  • When outliers dominate. MSE squares errors, so extreme outliers have enormous influence. One absurdly-priced mansion can completely skew your model for average homes.
  • When features are correlated (multicollinearity). If two features are nearly identical (e.g. size in m² and size in ft²), the model can't distinguish their individual contributions. Weights become unstable.
  • When you're predicting a category. Linear regression can predict any number including negative values and values above 1. For probabilities (which must be 0–1), you need logistic regression — tomorrow's topic.
🔗 Connection to Modern AI

The output layer of almost every neural network — including the language models powering ChatGPT — is a linear layer. The "head" of a classification model, the "projection" layer of a transformer, the value prediction in reinforcement learning — all linear. Linear regression isn't a relic. It's the fundamental building block everything else stacks on top of.

06 — A Thought to Sit With

💬 For You to Consider

Linear regression finds the line that minimizes average error. But average error treats all mistakes equally — a 10% error on a $100,000 house and a 10% error on a $10,000,000 house cost the same in MSE.

When might "average error" be the wrong thing to minimize? What would you optimize instead?

🎯 Interview Questions
  1. What does R² mean and what values indicate a good model?
    R² measures the proportion of variance in the target explained by the model. Values near 1.0 are good; near 0.0 means the model is no better than predicting the mean. Context matters — R²=0.7 is excellent for some domains, poor for others.
  2. What are the main assumptions of linear regression?
    Linearity (relationship between features and target is linear), independence of errors, homoscedasticity (constant error variance), normality of residuals, and no multicollinearity.
  3. Why is MSE preferred over MAE in many cases?
    MSE penalizes large errors more heavily due to squaring, making it differentiable everywhere (useful for gradient descent), and more sensitive to outliers. MAE is more robust to outliers and easier to interpret but not differentiable at zero.
  4. What is multicollinearity and how do you detect it?
    Multicollinearity occurs when two or more features are highly correlated, making individual coefficients unstable and hard to interpret. Detected using the Variance Inflation Factor (VIF) or correlation matrices.
  5. How does regularization (Ridge/Lasso) help linear regression?
    Regularization adds a penalty term to the loss function that discourages large weights, preventing overfitting. Ridge (L2) shrinks all weights. Lasso (L1) can zero out weights, performing automatic feature selection.
